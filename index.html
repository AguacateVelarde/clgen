Welcome to the CLgen documentation.

Requirements¶

  • Linux (x64) or OS X.
  • GCC > 4.7 or clang >= 3.1.
  • GNU Make > 3.79.
  • Python 2.7 or >= 3.4.
  • zlib >= 1.2.3.4.
  • libhdf5 >= 1.8.11.

Optional, but highly recommended:

  • OpenCL == 1.2.
  • An NVIDIA GPU and CUDA >= 6.5.

Installation¶

Download and unpack the latest CLgen release:

$ wget https://github.com/ChrisCummins/clgen/archive/0.0.11.tar.gz -O clgen-0.0.11.tar.gz
$ tar xf clgen-0.0.11.tar.gz
$ cd clgen-0.0.11

Compile and build CLgen using:

$ ./configure
$ make
$ sudo make install

This may take some time (over an hour). If you encounter any problems, please read the deailed build instructions and consider opening a bug report.

(Optional) Run the test suite using:

$ make test
